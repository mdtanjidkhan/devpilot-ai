# 🚀 DevPilot AI

**AI-powered software planning platform that transforms project ideas into complete development blueprints.**

DevPilot AI helps developers, students, freelancers, and startup founders turn simple software ideas into structured development plans using AI. Instead of spending hours planning a project, users can generate software architecture, database schema, API endpoints, UI pages, development roadmaps, and professional documentation within minutes.

---

## ✨ Features

* 🤖 AI-powered software planning
* 📋 Generate project overview and core features
* 👥 Automatic user role suggestions
* 🗄️ MongoDB database schema generation
* 🔗 REST API endpoint generation
* 📁 Frontend & backend folder structure suggestions
* 🎨 UI page planning
* 🛣️ Development roadmap generation
* 📄 Professional README generation
* 💾 Save AI-generated blueprints
* 📜 AI generation history
* 🔍 Project search
* 🌙 Light & Dark mode
* 🔐 Secure authentication with Better Auth
* 📱 Fully responsive dashboard

## 🚀 Installation

### Clone Repository

```bash
git clone 
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env.local` file for the frontend and a `.env` file for the backend.

Example:

```env
MONGODB_URI=
BETTER_AUTH_SECRET=
BETTER_AUTH_URL=
JWT_SECRET=
GEMINI_API_KEY=
```
